# `updis`
Little tool to update Discord on Debian-based systems.
![VERY ANNOYING IMAGE](https://i.stack.imgur.com/TwcBH.png)
## Installation
```bash
git clone https://github.com/rcroughs/updis
cd updis
sudo make install
```
## Usage
```bash
updis
```
AND BOOM YOU'RE DONE
